
# Sales Performance Analysis Project

## Objective
Analyze retail sales data to identify revenue trends, top-performing products, and regional performance.

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Sales trends analysis
- Category and region-wise performance
- Data visualization

## Dataset
Superstore Sales Dataset (Kaggle)

## Outcome
Generated insights to help understand sales patterns and business performance.
